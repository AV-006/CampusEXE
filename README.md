# CampusConnect

This is a Next.js application designed to be a central hub for campus life, connecting students and faculty.

## Features

- **Resources Hub**: Teachers can share notes, presentations, and other materials.
- **Interactive Forum**: A gamified system to encourage student participation with a leaderboard.
- **Anonymous Feedback**: An AI-powered system for students to provide anonymous feedback to faculty.
- **Campus Services**: A one-stop portal for library, canteen, mess, and gym information.
- **Events & Exams**: A calendar for all campus events and exam schedules.
