# CampusExe

This is a Next.js application designed to be a central hub for campus life, connecting students and faculty through a multi-agent AI system.

## Features

- **AI Tutor & Resources Hub**: An AI agent that provides personalized learning paths and resources, collaborating with other agents to identify students who need help.
- **Interactive Forum**: A gamified system to encourage student participation with a leaderboard.
- **Anonymous Feedback**: An AI-powered system for students to provide anonymous feedback to faculty.
- **Campus Services Agent**: A conversational AI that provides real-time information about the library, canteen, mess, and gym.
- **Events & Exams**: A calendar for all campus events and exam schedules.
