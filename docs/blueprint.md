# **App Name**: CampusConnect

## Core Features:

- Resources Hub: Teachers can upload and manage resources (notes, PPTs, books) for students.
- Interactive Forum: Points-based system to encourage student participation, with a leaderboard.
- Anonymous Feedback: Students can provide anonymous feedback to teachers through predefined questions. An AI tool generates a summary report for the faculty.
- Campus Services Portal: Unified interface for library resources, canteen/mess menus, and gym info, managed by an agent. Includes a proposal system for menu changes.
- Events and Exams Calendar: Centralized events and exams schedule, managed by various campus clubs. Students can volunteer, register, and provide suggestions for events/exams within the interface.

## Style Guidelines:

- Primary color: Orange Peel (#FFA500) for a warm and inviting feel.
- Background: Cream (#FFF6E5) as a background to create a clean aesthetic.
- Accent color: Dark Teal (#007C91) for calm contrast, use for section dividers or hover states.
- Body and headline font: 'PT Sans', a humanist sans-serif that blends a modern look with a touch of personality for both headings and body text.
- Use clear, friendly icons relevant to each section of the application (e.g., books for resources, speech bubbles for the forum, etc.).
- Employ a clean, card-based layout for displaying information, ensuring a user-friendly and easily navigable interface.
- Incorporate subtle hover effects on interactive elements to enhance user experience.
